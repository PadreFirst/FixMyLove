# Fix My Love — Telegram-бот психологической помощи в отношениях

AI-бот на базе Google Gemini, который помогает людям разбираться в ситуациях в отношениях. Работает по протоколам КПТ, не заменяет психолога.

## Стек

- **Python 3.11+**
- **aiogram 3.x** — Telegram Bot API
- **MongoDB + motor** — хранение профилей и сессий
- **Google Gemini API** — Flash-Lite, Flash, Pro
- **APScheduler** — крон-задачи (дневник, закрытие сессий)

## Быстрый старт

### 1. Зависимости

```bash
pip install -r requirements.txt
```

### 2. Переменные окружения

Заполни `.env` на сервере:

| Переменная | Описание |
|-----------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather |
| `GEMINI_API_KEY` | API-ключ Google AI Studio |
| `MONGODB_URI` | URI подключения к MongoDB (default: `mongodb://localhost:27017`) |
| `MONGODB_DB_NAME` | Имя базы данных (default: `fixmylove`) |

### 3. MongoDB

Локально:
```bash
mongod --dbpath /data/db
```

Или используй [MongoDB Atlas](https://cloud.mongodb.com/) (бесплатный tier).

### 4. Запуск

```bash
python main.py
```

## Архитектура

### Pipeline обработки сообщений

1. **Нормализация** — текст/голос/фото → текст
2. **Минимальный контекст** — загрузка из БД
3. **Классификация** — сначала правила, потом Flash-Lite
4. **Роутинг** — по типу сообщения
5. **Полный контекст** — загрузка для сессии
6. **Диагностика** — Flash-Lite → JSON
7. **Формирование промпта** — системный + данные + методика
8. **Генерация** — выбранная модель
9. **Отправка** — ответ пользователю
10. **Обновление БД** — асинхронно, не блокирует

### Модели Gemini

| Модель | Когда |
|--------|-------|
| Flash-Lite | Служебные задачи (классификация, диагностика, факты) |
| Flash | Дневник, админ, тренажёр, продолжение сессий |
| Flash + thinking | Новые сессии, острые ситуации, рефлексия |
| Pro | Кризис, первая сессия, длинная рефлексия |
